# Credits

This starter builds on work from several projects:

- **SKALE Network** ([skale.space](https://skale.space)) — gasless EVM L1 this starter targets. See [SKALE's x402 cookbook](https://docs.skale.space/cookbook/x402/accepting-payments).
- **DirtRoad Foundation** ([facilitator.dirtroad.dev](https://facilitator.dirtroad.dev)) — runs the x402 facilitator that settles SKALE payments.
- **PayAI Network** — forked from [`PayAINetwork/x402-hono-starter`](https://github.com/PayAINetwork/x402-hono-starter). Licensed Apache-2.0.
- **Coinbase** — original [`coinbase/x402`](https://github.com/coinbase/x402) protocol and `@x402/hono` middleware.
- **AgentTax** ([agenttax.io](https://agenttax.io)) — US sales tax calculation API used to record compliance on every settled transaction.

Full tax integration guide: [agenttax.io/integrations/skale](https://agenttax.io/integrations/skale)
